キレイン プライバシーポリシー

あなたの個人情報の保護について。

2026年6月3日 制定

1はじめに

Kirein（以下「本サービス」）は、ユーザーのプライバシーを尊重します。本プライバシーポリシーは、本サービスがユーザーの個人情報をどのように取得・利用・保護するかについて説明しています。

2取得する情報

2.1 ユーザーが直接入力する情報

  • ニックネーム：アカウント作成時に入力
  • 投稿内容：店舗評価・コメント
  • 投稿日時：自動記録

2.2 自動取得される情報

  • ブラウザ情報：閲覧中のブラウザ種類・バージョン
  • アクセス情報：ページ訪問時刻・滞在時間
  • デバイス情報：OS（iOS / Android / Windows など）
📍 位置情報について： 本サービスは GPS・位置情報を取得しません。店舗検索は手動入力またはリスト選択です。

3情報の保存場所

3.1 ローカルストレージ

現在、ユーザーの情報はブラウザの LocalStorageに保存されます：

  • ニックネーム・ポイント・Lv
  • 投稿履歴・評価内容
  • ログイン状態

これはユーザーのデバイス内のみに保存され、サーバーには送信されません。

3.2 将来のクラウド移行

将来、サーバー側データベースへの移行を検討する場合、ユーザーに事前に通知し、同意を求めます。

4情報の利用目的

  • サービス提供：投稿機能・ポイント計算・Lv 判定・マイページ表示
  • 分析・改善：ユーザー行動の匿名分析（投稿数・平均評価など）
  • セキュリティ：不正利用の検知・防止
  • お知らせ：サービス更新・重要なお知らせ（将来的にメール配信を追加する場合）

ユーザーの個人を特定できる情報を、第三者に販売・共有することはありません。

5投稿データの取り扱い

5.1 公開される情報

  • 肯定的な投稿（★5）のみ、店舗名とともに本サイトに掲載されます。
  • 投稿者のニックネーム・個人情報は一切公開されません。

5.2 非公開情報

  • 否定的な評価（★4以下）は、該当店舗にのみ匿名で届きます。
  • 一般ユーザーには表示されません。

5.3 投稿の削除について

一度投稿された内容は、削除・修正申請には応じられません（ブラウザの LocalStorage からは手動で削除可能）。投稿する際は慎重にしてください。

6第三者サービスの利用

6.1 Google Places API

本サービスは、店舗情報の取得に Google Places API を使用しています。

  • Google のプライバシーポリシー：https://policies.google.com/privacy
  • API 使用に伴う店舗情報（名前・住所・評価）は Google のプライバシーポリシーに従います。

6.2 将来的な外部連携

将来、SendGrid（メール配信）や Stripe（決済）などを導入する場合、それぞれのプライバシーポリシーに従います。事前に通知します。

7Cookie について

本サービスは Cookie を使用しません。 LocalStorage のみでユーザー情報を保存します。

8セキュリティ

  • ユーザーの情報は、ブラウザの LocalStorage に保存されます（暗号化されていません）。
  • 共有デバイスを使用している場合、他のユーザーが同じブラウザからアクセスすると、データが見える可能性があります。
  • セキュリティを最大化するため、プライベートブラウジングモードの利用を推奨します。
⚠️ 重要： LocalStorage は基本的に暗号化されていません。公共 WiFi や共有デバイスでの利用は避けてください。

9データの保持期間

  • ユーザーのデータは、本サービスを利用する限り保持されます。
  • ブラウザの LocalStorage をクリアすると、すべてのユーザーデータが削除されます。
  • サービス終了時、ユーザーのすべてのデータが失効します。

10ユーザーの権利

  • アクセス権：ユーザーはいつでも自分の投稿履歴を確認できます（マイページ）。
  • 削除権：ブラウザの LocalStorage をクリアすることで、自分のデータを削除できます。
  • 異議申し立て：本ポリシーに関する質問・異議は、お問い合わせフォームからご連絡ください。

11ポリシーの変更

本プライバシーポリシーは予告なく変更されることがあります。重大な変更の場合は、事前にサイト上で告知します。
